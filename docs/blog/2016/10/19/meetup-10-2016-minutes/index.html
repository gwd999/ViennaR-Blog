Meetup 10-2016 Minutes

The October Meetup was hosted by Roland Boubela who gave an excellent talk about the new sparklyr package. sparklyr provides an easy-to-use interface to Apache Spark and can even be accessed using the latest and greatest version of RStudio (preview release). It also covered a comparison with the existing SparkR package.
